## 🚀 Deployment

This site is deployed using GitHub Pages and is automatically updated when changes are pushed to the repository.

### To Enable GitHub Pages:
1. Go to Repository Settings
2. Scroll to "Pages" section
3. Select "Deploy from a branch"
4. Choose "main" branch
5. Click Save

The site will be live at: `https://crowdmineinvestment1.github.io/crowdmine-investment-/`
